### 访问权限修饰符



- java提供了4种访问权限修饰符，访问权限从大到小：public、protect、default、private

| 权限修饰符 | 同一个类 | 同一个包 | 不同包的子类 | 不同包的非子类 |
| ---------- | -------- | -------- | ------------ | -------------- |
| Private    | √        |          |              |                |
| Default    | √        | √        |              |                |
| Protected  | √        | √        | √            |                |
| Public     | √        | √        | √            | √              |
